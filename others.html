How to Use This Player

To embed any YouTube video, simply add this HTML:

<div data-video-id="YOUTUBE_VIDEO_ID"></div>

Example:

<div data-video-id="dQw4w9WgXcQ"></div>

Requirements:

  • Include player.js in your page
  • Add element with data-video-id attribute
  • That's it! The player auto-initializes
