Setup Required

To use this reels system, you need to create a JSON file with your video data.

{
    "reels": [
        {
            "id": 1,
            "title": "Physics Concept Explained",
            "description": "Short explanation of Newton's Laws",
            "videoId": "dQw4w9WgXcQ",
            "category": "Physics",
            "creator": "Science Academy",
            "creatorAvatar": "https://example.com/avatar.jpg",
            "date": "2024-01-15",
            "likes": 1500,
            "comments": 45,
            "shares": 120
        }
    ]
}

Save this as reels-data.json in the same folder as this HTML file.
